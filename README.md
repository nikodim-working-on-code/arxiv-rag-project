# arXiv RAG Project

AI-powered Research Assistant for arXiv Papers using RAG

## Tech Stack

- **Embeddings:** nomic-ai/nomic-embed-text-v1.5
- **Vector DB:** ChromaDB
- **Data Source:** arXiv API
